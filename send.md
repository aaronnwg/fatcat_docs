# Send

Transfer tokens from your FatCat wallet to any external Solana wallet.

### Via Telegram bot

1. Tap **Send** from the main menu.
2. Enter the destination wallet address.
3. Select the token and amount.
4. Confirm.

### Via Mobile App

1. Tap the **Send** tab.
2. Enter the destination address, token, and amount.
3. Confirm.

{% hint style="warning" %}
Solana transactions are irreversible. Always double-check the address before confirming.
{% endhint %}

{% hint style="info" %}
You need a small amount of SOL in your wallet to cover the network fee (~0.000005 SOL).
You can send any token in your wallet — SOL, USDC, wETH, wBTC, or any SPL token.
{% endhint %}
