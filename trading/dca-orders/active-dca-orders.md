# Active DCA Orders

**Button:** `🗓️ Active DCA Orders` (DCA Menu)

Open `⏰ DCA` → `🗓️ Active DCA Orders`.

* View progress (completed executions vs remaining).
* Cancel an active DCA to stop future executions.

{% hint style="info" %}
Cancelling stops **future** executions only.

Any already-executed swaps stay executed.

Any fees that have already been collected will not be refunded upon cancelling.
{% endhint %}

You can also view and manage DCA orders on Jupiter since they live on Jupiter’s recurring program:

<https://jup.ag/recurring>

### What you’ll see

* A list of your currently active DCA orders.
* Per-order details like token pair, size, frequency, and next execution time.
* Status/progress.
* Controls to cancel active DCA orders.

### Cancelling a DCA order

You can cancel an active DCA at any time from the Open Orders menu or the FatCat Mobile App.

Remaining uncommitted funds are returned to your wallet.

Orders that have already executed cannot be reversed.

### Troubleshooting

* **My active list is empty:** confirm you’re connected to the same wallet that created the DCA.
* **I cancelled, but it still shows:** refresh after a short delay. Indexing can lag during congestion.
