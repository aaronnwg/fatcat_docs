# DCA Order History

**Button:** `📔 DCA Order History` (DCA Menu)

Open `⏰ DCA` → `📔 DCA Order History`.

Use DCA Order History to review your completed and cancelled DCA orders.

### What you can do

* Confirm whether a DCA completed or was cancelled.
* Review past DCA outcomes.

### What you’ll see

* Completed DCA orders.
* Cancelled DCA orders (if any).

{% hint style="info" %}
DCA orders live on Jupiter’s recurring program.

You can also review them on: <https://jup.ag/recurring>
{% endhint %}

### What the list includes

Expect each entry to include most of these fields:

* Token pair.
* Status: **Completed** or **Cancelled**.
* Size per execution and frequency.
* Executions completed (and the final total).
* Timestamps (created / last execution / end).

{% hint style="info" %}
If your history looks empty, confirm you’re viewing the same wallet you used for the DCA.
{% endhint %}
