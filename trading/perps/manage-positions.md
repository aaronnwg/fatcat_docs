# Manage positions

Manage Perp positions from `🎯 Active Positions`.

Path: `Main Menu` → `🎯 Active Positions`.

### View positions

Open `🎯 Active Positions` to see:

* Asset + direction (long/short)
* Leverage
* Current PnL

Tap a position to open its controls.

### Position controls

| Button                              | Function                         |
| ----------------------------------- | -------------------------------- |
| `🔴 Close Position`                 | Close position (partial or full) |
| `🛑 Stop Loss` / `🛑 SL Active ✓`   | Set or manage stop loss          |
| `🎯 Take Profit` / `🎯 TP Active ✓` | Set or manage take profit        |
| `🔄 Refresh Position`               | Update position data             |

### Close a position

1. Click `🔴 Close Position`.
2. Click `💯 Close Entire Position` or `📝 Close Partial Position` and then select a percentage (25%, 50%, 75%) or type a custom percentage.
3. Review and approve.

### Stop loss / take profit

1. Click `🛑 Stop Loss` or `🎯 Take Profit`.
2. Select a percentage (15%, 50%, 75%, 100%) or type a custom percentage for the amount to close at trigger price.
3. Enter the trigger price.
4. Review and approve.

### History

Open `📚 Perp History` to view completed/closed positions.

See: [Perp History](https://docs.fatcatbot.io/trading/perps/perp-history).
