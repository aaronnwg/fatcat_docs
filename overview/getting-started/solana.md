# Solana

FatCat trades on the Solana blockchain. Through FatCat you can trade Solana tokens and on-chain Perpetual contracts.

Most actions need a small SOL balance.

### Why you need SOL

* **Network fees:** every on-chain action costs a small amount of SOL.
* **Trading actions:** swaps and order actions are on-chain.
* **Smooth execution:** an insufficient SOL balance can cause failed or stuck actions.

### Where to get SOL

Use one of these options:

* A centralized exchange (Coinbase, Binance, Kraken, etc.)
* Another onramp provider supported in your country
* Transfer SOL from another Solana wallet you control

{% hint style="warning" %}
Only send SOL using the **Solana** network to a Solana address. Sending on the wrong network can result in permanent loss.
{% endhint %}

### Common mistakes to avoid

* Sending SOL on the wrong network (for example, an exchange “SOL” token on another chain).
* Copy/paste errors on the wallet address.
* Running your SOL balance to zero, then being unable to sell/cancel.

### Safety notes

* On-chain transactions are final. There are no chargebacks.
* Always verify the address and network before sending.
* FatCat does not sell SOL. We only connect to your wallet.
